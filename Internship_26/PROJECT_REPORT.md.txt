# Final Project Report: 4-Way Traffic Light Controller ASIC Design Flow

## 1. Project Overview
This project presents the design, verification, and physical implementation of a synthesizable **4-Way Traffic Light Controller**. The control logic is designed using Verilog HDL based on a finite state machine (FSM) architecture that governs standard green-yellow-red scheduled time intervals for an intersection. 

The entire back-end implementation follows a professional ASIC development lifecycle using the **Cadence Tool Suite**, spanning from logical synthesis to physical floorplanning, power distribution networks, placement optimization, clock distribution tracking, track routing, and design rule signoff.

---

## 2. Design Methodology
The development architecture is structured into 8 distinct engineering milestones, isolating design entry from physical backend layout configurations:

1. **Design Entry & Functional Simulation:** Behavioral modeling and comprehensive testbench execution to assert state transition parameters and reset conditions.
2. **Logic Synthesis:** Technology mapping of behavioral Verilog code into structural gate-level components using area and timing constraints.
3. **Floorplanning:** Initializing the physical die boundary dimensions, target core utilization ratios, and constraining peripheral I/O pad placements.
4. **Power Planning:** Structural layout design of uniform global VDD/VSS supply rings and interleaved current distribution stripes.
5. **Standard Cell Placement:** Automated timing-driven row assignment for all synthesized logic gates to optimize timing slack and wire length.
6. **Clock Tree Synthesis (CTS):** Building dynamic, balanced buffer networks targeting sequential components to drastically reduce total clock skew.
7. **Global & Detail Routing:** Finalizing signal track connectivity across processing metal layers while optimizing metal track density.
8. **Physical Verification & Signoff:** Running automated Design Rule Checking (DRC) to guarantee geometric and layout print compliance.

## 5. Summary of Results
* **Functional Integrity:** Waveform traces confirm standard cyclic state scheduling (Green -> Yellow -> Red) for all four directions without any sequencing errors.
* **Timing Closure:** Post-CTS and post-routing evaluations confirm positive setup and hold margins across all operational logic path vectors.
* **Physical Signoff:** Layout checks confirm 100% compliance with geometric manufacturing rules, resulting in **zero** DRC layout violations.
